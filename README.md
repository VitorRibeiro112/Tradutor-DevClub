# Tradutor DevClub

Projeto desenvolvido para traduzir textos de forma simples.

## Tecnologias usadas

- JavaScript

- HTML

- CSS

## Como usar

1. Clone o repositório  

2. Abra o arquivo `index.html` no navegador  

3. Digite o texto que deseja traduzir e clique em Traduzir
